Travel & Tourism Website 🌍✈️

A simple Travel and Tourism Website built using HTML.
This project contains multiple pages including a home page, login page, places page, and booking page.

Features
.Attractive Home Page
.User Login Page
.Tourist Places Information Page
.Travel Booking Page
.Simple and beginner-friendly HTML structure
.Easy to customize

Pages Description

1. Home Page (index.html)

Main landing page of the website
Contains navigation bar, welcome section, and featured destinations

2. Login Page (login.html)
   
User authentication form
Includes username/email and password fields

3. Places Page (places.html)
   
Displays famous tourist destinations
Includes images and short descriptions

4. Booking Page (booking.html)
   
Allows users to book travel packages
Contains booking form with user details
